CSS Selectors

The Position Property

Selector Description
static An element is always positioned according to the normal flow of the page
relative An element with position: relative; is positioned relative to its normal position.
fixed Element is positioned relative to the viewport, it always stays in the same place even if scrolled.
absolute An element with position: absolute; is positioned relative to the nearest positioned ancestor
sticky An element with position: sticky; is positioned based on the user's scroll position.
